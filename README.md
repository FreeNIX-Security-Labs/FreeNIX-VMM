## Installation
To install FreeNIX-VMM, type these commands:
```
$ git clone git://github.com/FreeNIX-Security-Labs/FreeNIX-VMM/
$ cd FreeNIX-VMM
$ chmod 755 configure.sh
# ./configure.sh
# ./freenix-vmm.sh
```
